# FilterYou 项目开发规划 (Roadmap)

本文档记录了 FilterYou 的长期开发规划和待办功能清单，旨在为后续的迭代提供清晰的指导。

## 核心愿景
打造一款简约、清新、美观且功能完备的 Android 开源电话拦截工具。

---

## 📅 当前状态：阶段 0 (已完成)
- [x] 基础拦截框架 (CallScreeningService)
- [x] 基于 Room 的规则持久化
- [x] 现代化 UI 重构 (Material 3, 薄荷绿主题)
- [x] Hilt 依赖注入迁移
- [x] 多种拦截策略支持 (包含、开头、结尾、正则、地区识别)

---

## 🚀 阶段 1：核心功能补完 (进行中)
本阶段目标是补齐市面主流拦截应用必备的基础功能。

- [x] **白名单管理**
    - 支持添加“白名单”规则，优先级高于黑名单，确保重要号码永不拦截。
- [ ] **通讯录保护模式**
    - 增加“仅允许联系人”全局开关。
    - 技术实现：集成 `READ_CONTACTS` 权限并与系统通讯录对比。
- [ ] **拦截反馈系统**
    - 拦截成功后的静默通知提醒。
    - 点击通知快速进入日志详情。

---

## 🎨 阶段 2：智能化与体验升级
提升应用的“聪明度”和视觉表现。

- [ ] **可视化数据报表**
    - Dashboard 引入饼图/柱状图，展示不同拦截策略的命中分布。
- [ ] **空号/忙音伪装**
    - 调研并实现拦截后的响应模式（如返回忙音、空号提示等）。
- [ ] **日志自动维护**
    - 支持设置“日志保留时长”（如 7天、30天），过期自动清理数据库。

---

## 🛠️ 阶段 3：数据互通与高级功能
针对极客和高级用户的进阶能力。

- [ ] **规则导出与导入**
    - 支持 JSON 格式导出规则，方便换机备份或共享规则集。
- [ ] **AI 语音代接 (长期目标)**
    - 集成语音转文字技术，为用户提供智能助理接听服务。
- [ ] **开源骚扰库集成**
    - 调研并接入公开的骚扰电话数据库，减少用户手动配置成本。

---

## 📝 维护说明
任何对此项目进行开发的 Agent 或开发者，应在完成相应功能后更新此文档的状态。
